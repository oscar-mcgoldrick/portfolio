Magic Mike's Mystical Emporium

This project was a 1-day build in a group of four, the same week we learned about API integration. The project has 2 main features, a fortune teller where the user is read their 'fortune' based on their birthday using the OpenAI API. We used the same API to give the user a 'soulmate' in the other feature.

This was a fun, light-hearted way to show off some complex understandings at the week 5 mark of our 10 week bootcamp. I was responsible for state management & passing states to child components, as well as helping with the API integration.
