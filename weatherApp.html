Weather App

This was a personal project I worked on throughout the duration of the bootcamp. There is a database of Cities, which a user can retrieve real-time weather information using an open API based on the latitude and longitude information. The other feature is the search bar, a user can search for any city and be provided the same weather information. This uses the same API as the first feature, and additionally makes a call to retrieve the latitude and longitude information based on the city name.

My drive for creating this project was that I wanted a simple outlet to try new things in code, and any subsequent ideas I could build on top of it.
